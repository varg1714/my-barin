# 1. Linux 大纲

- Linux 常用命令
	- Tail
	- Cat
		- 显示整个文件：cat filename
		- 创建文件：cat > filename
		- 合并文件：cat file1 file2 > newfile
		- 日志追加：cat -n file1 > file2
		- 清空文件：cat : >file
	- More
		- 向下滚动一行：enter
		- 向下滚动一屏：space / ctrl + f
		- 返回上一屏：ctrl + b
	- Sed
	- Less
		- 到文件尾部：shift + G
		- 搜索：? Xxx，向上查找：n，向下查找：shift + n
