
# 1. 按时间排序

```dataview

TABLE date(create,"yyyy-MM-dd HH:mm") as 创建时间,round((date(now) - date(create,"yyyy-MM-dd HH:mm")).day,1) as 遗留天数
FROM "阅读中"
WHERE file.name != "__阅读目录"
SORT create asc,file.name asc

```

# 3. 已阅读待整理

```dataview

TABLE date(create,"yyyy-MM-dd HH:mm") as 创建时间,round((date(now) - date(create,"yyyy-MM-dd HH:mm")).day,1) as 遗留天数
```
