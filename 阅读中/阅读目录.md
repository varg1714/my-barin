## 1. 阅读中

```dataview

TABLE tags as 标签,dateformat(date(create,"yyyy-MM-dd HH:mm"),"yyyy-MM-dd") as 创建时间,
    max(0,round((date(now) - date(create,"yyyy-MM-dd HH:mm")).day,0)) as 遗留天数
FROM "阅读中/文章列表"
WHERE read != true
SORT create desc,file.name asc

```

## 2. 已阅读待整理

```dataview

TABLE tags as 标签,dateformat(date(create,"yyyy-MM-dd HH:mm"),"yyyy-MM-dd") as 创建时间,max(0,round((date(now) - date(create,"yyyy-MM-dd HH:mm")).day,0)) as 遗留天数
FROM "阅读中/文章列表"
WHERE read = true and knowledge != true
SORT create desc,file.name asc

```

## 3. 知识化

```dataview

TABLE tags as 标签,dateformat(date(create,"yyyy-MM-dd HH:mm"),"yyyy-MM-dd") as 创建时间,
dateformat(knowledge-date,"yyyy-MM-dd") as 知识化时间
FROM "阅读中/文章列表"
WHERE knowledge = true
SORT knowledge-date desc,create desc,file.name asc

```

## 4. 总结化

```dataview
TABLE source as 原文章,dateformat(file.ctime,"yyyy-MM-dd") as 创建时间
FROM "阅读中/阅读总结"
SORT file.ctime desc,file.name asc
```
